# SCU Telegram Bot

## Descripción

Bot de telegram para consultar el menú de los comedores de la Universidad de Granada

## Utilizaremos

- Typescript como lenguaje de programación y Node.js como _runtime_
- Jest como framework de pruebas
- API de Telegram
